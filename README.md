# Nacl Music

Um projeto criado com fins educativos e para treino de linguagem. Consiste em um conjunto de páginas interligadas e funcionais.

## Objetivos
* Prática da estrutura de rotas e sua aplicação.
* Manipulação de banco de dados(MySQL) utilizando o jinja.

## Tecnologias
* Linguagem: Python e HTML.

## Bibliotecas
* Jinja.

## Instalação e Uso
1. Clone o repositório ou baixe os arquivos.
2. O projeto não exige instalações externas, basta apenas executar o arquivo "app.py" e o banco de dados.
